Pull e push

Traz e leva os arquivos do repositório para sua máquina

Pull

git pull origin(repositório remoto) main(branch)

Push

git push origin(repositório remoto) main(branch)

mais comandos

Repositório remoto

git remote add nome-repositorio(padrão: origin) caminho-repositorio - adiciona local remoto

git clone servidor nome-da-pasta-que-fica-nesse-local

Branchs (ramos)

git branch nome-do-branch - cria um novo ramo

git branch - mostra os ramos existentes

git checkout nome-do-branch - muda de ramo

git revert hash-do-commit - desfaz um commit

git stash - salva alterações que não foram adicionadas nem commitadas
